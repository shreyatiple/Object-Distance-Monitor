# Object Distance Monitor
Object Distance Monitor is a real-time computer vision system that detects objects and measures the distance between them to determine whether there is a gap or no gap. It helps ensure proper placement and alignment in applications like manufacturing, packaging, and safety monitoring, improving accuracy and quality control.
